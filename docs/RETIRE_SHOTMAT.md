# ShotMat listing — Withdraw & Resolution Center

PartyRound yeni listing olarak gönderilecek. Eski ShotMat kaydı için:

## Resolution Center reply (kopyala-yapıştır)

App Store Connect → ShotMat → **App Review** → reddedilen mesaja **Reply**:

```
Thank you for the feedback on Guideline 4.3(b).

We understand the concern. We are not resubmitting ShotMat under the same concept.

We have created a new app, PartyRound (com.kamoteakko.partyround), repositioned as an offline party question game with optional scoring or questions-only mode, bilingual support, custom questions, and on-device data only — without drinking-game branding.

We will submit PartyRound as a new App Store listing. The ShotMat submission can remain rejected / be removed from review.
```

## ShotMat listing’i ne yapmalı?

### Seçenek A — Leave rejected (en basit)
Hiçbir şey yapma. ShotMat **Rejected** kalır; yeni PartyRound ayrı app olur.

### Seçenek B — Remove from Sale / cancel
1. App Store Connect → ShotMat → **Pricing and Availability**
2. Availability’yi kapat veya app’i satıştan kaldır (yayında değilse zaten görünmez)
3. İstersen **App Information** notuna “Replaced by PartyRound” yaz

### Seçenek C — Delete app (kalıcı)
Apple nadiren silmeye izin verir; genelde rejected app’leri olduğu gibi bırakmak yeterli.

## Yapma

- ShotMat **1.0** için **Update Review / resubmit** yapma — aynı 4.3(b) riski yüksek
- Aynı `com.shotmat.app` bundle ile “sadece isim değişikliği” yeterli değil; PartyRound **yeni App ID** kullanır
