# Apple Developer Account Setup

PartyRound için mevcut Developer Program hesabı kullanılabilir.

1. [developer.apple.com](https://developer.apple.com) → Account
2. **Agreements** → Paid Applications aktif olmalı
3. Yeni App ID: `com.kamoteakko.partyround` + In-App Purchase
4. Detaylı ASC adımları: [PARTYROUND_ASC_SETUP.md](./PARTYROUND_ASC_SETUP.md)
5. Eski ShotMat: [RETIRE_SHOTMAT.md](./RETIRE_SHOTMAT.md)
